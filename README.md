# library_gui

Простое приложение книжного магазина на Kivy.

## Установка

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/theJorDea/library_gui.git
   cd library_gui
   ```
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

## Запуск

```bash
python app.py
```
